# Install Commands

Copy-paste install for any skill in this registry. Requires `skills-master-mcp` — run `/setup-skills-master-mcp` if not yet configured.

## 🔒 Security

```bash
# skillvet — run this first to vet any skill before installing
npx skills add "openclaw/skills" -s "skillvet" -a claude-code -y
```
